# DSP Web Clone (School Project)

- Built with React + Vite
- No app logic: structure and styling only
- Pages included: Home, NFT, Trading, Pool Data, Loan, Paper, Account Login

## Run

```bash
npm run dev
```

## Notes

- Navigation is hash-based : try `/#/in`, `/#/nft`, `/#/t`, `/#/pool`, `/#/lo`, `/#/paper`, `/#/n`.
