# BSP Web Clone (School Project)

This is a clean-room, static UI clone inspired by `https://www.bsp-web.app/` for a school project.

- Built with React + Vite
- No app logic: structure and styling only
- Pages included: Home, NFT, Trading, Pool Data, Loan, Paper, Account Login

## Run

```bash
npm run dev
```

## Notes

- This project is for learning and demonstration; avoid copying brand assets or claiming the original work as yours.
- Navigation is hash-based (similar to the original): try `/#/in`, `/#/nft`, `/#/t`, `/#/pool`, `/#/lo`, `/#/paper`, `/#/n`.
